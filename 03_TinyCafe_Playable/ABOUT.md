##[ TinyCafe Playable ]
→ 고양이 손님을 맞이하며 커피를 만들고, 카페를 운영하는 **카페 시뮬레이션 게임**

---

##* 게임을 하면서 느꼈던 재미 요소

1. **고양이(손님)에게 메뉴를 만들어 준다는 독특한 구성** — 손님이 사람이 아니라 고양이, 카페 직원이 생쥐라는 게 귀엽고 독특함
2. **모션 그래픽** — 돌체가 커피를 내리는 모션, 직원이 물을 떠서 머신에 넣는 모션처럼 캐릭터가 실제로 일하는 것처럼 움직이는 연출 (but 실제 게임처럼 자세히 구현하기 어려울 것 같음)
3. **현실감 있는 음료 제작** — 핸드드립, 에스프레소, 아메리카노를 실제 카페 도구, 유사한 레시피로 만드는 과정이 있어서 "진짜 카페와 매우 흡사한 느낌"
4. **전반적인 귀여움** — 비주얼, 캐릭터, UI 모두 귀여운 톤으로 통일되어 있어 오래 봐도 질리지 않음

이 네 가지 요소를 중심으로 구성하고자 함

---

##* 꼭 넣어야 할 필수 기능들

### 1. 고양이 손님 시스템
- **갈색 후드 고양이(Cat1)**: 핸드드립 선호
- **네이비 후드 고양이(Cat2)**: 에스프레소 선호
- 감정 상태 변화: 대기 → 슬픔 → 화남 → 음료 받는 중 → 만족

손님은 카페 창문 밖에서 걸어오고, 15초 안에 음료를 받지 못하면 화난 얼굴로 떠납니다.

### 2. 카페 메뉴 제작 (머신 + 도구)
머신 3종이 각자 다른 음료를 만들고, 실제 카페처럼 서로 다른 방식으로 작동합니다.

| 머신 | 음료 | 제작 방식 | 특징 |
|------|------|----------|------|
| 핸드드립 | 드립 커피 | 필터 + 드립 | 게임 시작부터 보유 |
| 에스프레소 | 에스프레소 | 압력 추출 + 크레마 | 200코인에 해금 |
| 물 펌프 | 아메리카노 | 물 + 에스프레소 | 800코인에 해금 |

각 머신은 진행도 바가 달려 있고, 생산이 완료되면 코인 슬롯(하단 바)에 음료 카드가 쌓입니다.

### 3. 캐릭터 모션
원작 게임만큼은 아니더라도 캐릭터에게 움직임 효과를 주기 위해 간단한 모션을 추가
- **돌체(Dolce)**: 카페 중앙 사다리 위에서 상시 대기하며, 손님이 주문하면 위아래로 통통 튀는 모션으로 "커피를 만드는" 분위기를 연출
- **에스프레소 직원(쥐)**: 왼쪽에서 좌우로 흔들리는 모션으로 상시 작업 중
- **물 펌프 직원(쥐)**: 오른쪽에서 회전하는 모션으로 컵에 물을 받는 동작 표현
- **고양이 손님**: 좌우에서 걷는 입장 모션 + 위아래 통통 튀는 대기 모션 + 감정별 PNG 교체

### 4. 코인으로 머신 업그레이드
"돈 모아서 더 좋은 기계 사고, 기계가 더 많은 돈을 버는" 게임의 핵심 루프입니다.
- 현재 각 머신마다 최대 50레벨까지 업그레이드 가능
- 업그레이드할수록 생산 속도와 수익이 증가
- 알바넷(채용 시스템)으로 직원을 고용하면 수익이 배수로 증가 (현재는 월드맵이 열리지 않아 치즈를 생산할 건물 존재 X → 장비당 2명 이상의 직원 추가 고용 불가)

## 게임 시스템 상세 설명

### 기본 게임 루프

```
머신이 자동으로 음료 생산
       ↓
코인 슬롯(하단 바)에 제조된 음료 쌓임
       ↓
카드 탭 or 10초 후 자동 수집 (고양이 발 등장 및 수거)
       ↓
손님 고양이가 창문 밖에 등장
       ↓
Brew 버튼 탭 → 음료 제작 중 (진행 바)
       ↓
서빙 완료 → 3배 보너스 코인 획득
       ↓
코인으로 머신 업그레이드 / 신규 머신 구매
```

### 자원 시스템

게임에는 4가지 자원이 있습니다.

| 자원 | 획득 방법 | 사용처 |
|------|----------|--------|
| ❤ 하트 | 미션 완료 | 미래 콘텐츠 해금 |
| 🪙 코인 | 머신 생산, 손님 서빙 | 머신 구매·업그레이드 |
| 🧀 치즈 | 목욕탕(Bathhouse) 자동 생산 | 알바넷 직원 채용 |
| 💎 젬 | 초기 지급 (5개) | 미래 가챠 시스템 |

### 손님 감정 시스템

손님이 창문 앞에 도착하면 기다리는 시간에 따라 표정이 바뀝니다.

```
0 ~ 7초  : 기쁨 (주문 대기)
7 ~ 12초 : 슬픔 (기다림에 지침)
12 ~ 15초: 화남 (곧 떠날 것 같음)
15초 초과 : 화난 얼굴로 퇴장 (코인 없음)
음료 받음 : 마시는 중 → 만족하며 퇴장
```

### 업그레이드 시스템

**머신 업그레이드**
- 코인 소모 → 레벨 +1
- 레벨당 수익 40% 증가
- 업그레이드 비용: `기본 비용 × 레벨 × 1.6`

**알바넷 (직원 채용)**
- 머신당 최대 5명 채용 가능
- 직원 1명당 수익 +100% 배율 적용
- 치즈로 채용 (첫 번째는 무료, 이후 지수적 증가)

### 미션 시스템

5개의 미션이 순서대로 주어지며 하트를 보상으로 줍니다.

| No. | 미션 | 목표 | 보상 |
|---|------|------|------|
| 1 | 손님 1명 서빙하기 | 1명 | ❤ +10 |
| 2 | 손님 10명 서빙하기 | 10명 | ❤ +20 |
| 3 | 핸드드립 머신 업그레이드하기 | 1회 | ❤ +25 |
| 4 | 에스프레소 머신 설치하기 | 1회 | ❤ +30 |
| 5 | 직원 1명 채용하기 | 1명 | ❤ +35 |
| 6 | 워터펌프 설치하기 | 1회 | ❤ +45 |
| 7 | 손님 30명 서빙하기 | 30명 | ❤ +50 |

## 화면 구성

### 메인 카페 화면

```
┌─────────────────────────┐
│ HUD (자원 표시 바)        │  ← 코인, 치즈, 하트, 젬
├─────────────────────────┤
│ 미션 진행 바              │
├─────────────────────────┤
│                         │
│   창문 존 (손님 등장 구역) │  ← 도시 배경 + 아치형 창문
│                         │
├─────────────────────────┤
│   카운터 분리선           │
├─────────────────────────┤
│                         │
│   카페 인테리어           │  ← 돌체 + 머신들 + 직원들
│   (핸드드립 / 에스프레소  │
│    / 워터펌프 / 목욕탕)   │
│                         │
├─────────────────────────┤
│ 코인 슬롯 바              │  ← 음료 카드 + 고양이 발 자동 수집
└─────────────────────────┘
```

### 스마트폰 (보조 메뉴)

인게임 스마트폰 UI로 추가 기능에 접근합니다.

- **시설**: 보유 머신 목록, 레벨 및 업그레이드 상세 정보
- **알바넷**: 직원 채용 화면 (드립팀/에스프레소팀/워터팀)
- **레시피**: 제조 가능한 음료 목록 및 설명
- **월드맵**: 미래 지점 선택 (준비 중)

## 현재 버전 범위 및 향후 계획

**현재 구현**
- 머신 4종 + 레벨 시스템
- 고양이 손님 2종 + 감정 시스템
- 알바넷 직원 채용
- 미션 7개
- 오프라인 보상 (최대 8시간)
- 모든 핵심 캐릭터 모션

**향후 계획**
- 월드맵 기반 지점 확장
- 신규 메뉴 추가
- 치즈 생산 건물 및 직원 설정
- 가챠 시스템 (젬 사용)
- 고양이 도감(Catbook)
