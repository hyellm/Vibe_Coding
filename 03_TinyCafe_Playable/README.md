# Tiny Cafe — Playable Game (Web)

Nanali Studios의 모바일 게임 "Tiny Cafe"를 참고해 React + TypeScript로 구현한 브라우저 플레이어블 아이들 게임입니다.
390×844px 모바일 뷰포트 기준으로 데스크톱 브라우저에서 플레이할 수 있습니다.

---

## 스크린샷

| 메인 카페 화면 | 업그레이드 팝업 | 브루 버튼 인터랙션 |
|:---:|:---:|:---:|
| 창문 존 + 카페 인테리어 | 슬라이드업 모달 | 고양이 손님 서빙 |

---

## 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| 프레임워크 | React 18 + TypeScript |
| 빌드 도구 | Vite 5 |
| 상태 관리 | Zustcdand 4 (+ persist 미들웨어) |
| 애니메이션 | Framer Motion 11 |
| 스타일링 | Tailwind CSS 3 |
| 저장소 | localStorage (자동 저장) |

## 핵심 게임 루프

```
장비 자동 생산 → 코인 슬롯 카드 생성 → 플레이어가 탭해서 코인 수집
                                    ↕
고양이 손님 등장 (창문 존) → 주문 말풍선 → 브루 버튼 탭 → 음료 서빙 → 보너스 코인 카드
```

1. **장비**가 자동으로 음료를 생산하면 하단 코인 슬롯 카드가 쌓입니다.
2. **코인 슬롯 카드**를 탭해서 코인을 수집합니다.
3. **고양이 손님**이 5~10초마다 창밖에서 랜덤 방향으로 등장해 음료를 주문합니다.
4. **브루 버튼**(녹색 주전자)을 탭 → 생산 진행 → 음료 서빙 → 손님이 음료를 마시고 퇴장 + 보너스 코인 카드.
5. 15초 내에 서빙 못하면 손님은 화난 표정(angry)으로 퇴장합니다.
6. 코인으로 장비를 **업그레이드**하면 생산 속도와 코인 단가가 올라갑니다.

## 고양이 손님 캐릭터

| 종류 | 선호 음료 |
|------|-----------|
| cat1 (갈색 후디) | 드립 커피 |
| cat2 (남색 후디) | 아메리카노 |

**대기 시간별 이미지 전환:**

| 상태 | 시점 | 이미지 |
|------|------|--------|
| smile | 등장 ~ 7초 / 서빙 완료 후 퇴장 | 😊 기본 표정 |
| sad | 7초 ~ 12초 | 😢 슬픔 |
| angry | 12초 ~ 15초(타임아웃) | 😠 화남 |
| drink | 음료 전달 직후 | 🍵 음료 마시는 동작 |

**손님 이동 방향 (랜덤 2가지):**
- 왼쪽 진입 → 화면 중앙 정지 → 오른쪽 퇴장
- 오른쪽 진입 → 화면 중앙 정지 → 왼쪽 퇴장

15초 내 서빙 못 하면 angry 이미지 유지 상태로 퇴장
